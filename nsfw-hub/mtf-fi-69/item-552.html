Единица снаряжения специалиста мобильной оперативной группы
ФИ-69 «Виагра» №-69-19-552
«CONTAIN THIS BITCH»

Описание особого снаряжения для проведения мероприятий по восстановлению условий содержания объектов класса «Возбужденный кетер»

Описание: устройство входит в состав особого снаряжения специалиста мобильной оперативной группы ФИ-69 «Виагра» (далее – специалист ФИ-69) для успешного проведения мероприятий по восстановлению условий содержания (далее – ВУС) объектов класса «Возбужденный кетер», в данном случае применяется для SCP-939-S-EX - «Со множеством голосов, и не только...».

Состав: устройство изготовлено из термостойкого силикона, не имеющего запаха. Данное свойство позволяет производить термическую обработку устройства для поддержания стерильности и предотвращения риска скапливания опасных бактерий. Кроме того, в состав добавлен элемент [ДАННЫЕ УДАЛЕНЫ], повышающий прочность и увеличивающий срок службы устройства в связи с его специфическим назначением.

Внешний вид: внешне устройство имеет фаллическую форму, воссозданную на основе данных, полученных при исследовании мужских половых органов существ [ДАННЫЕ УДАЛЕНЫ]. Для расширения функционала к устройству прилагаются несколько силиконовых насадок, надевающихся поверх устройства и снабженных различными выпуклостями для дополнительной сексуальной стимуляции объекта. Ближе к основанию имеется отметка «Contained Successfully», означающая рекомендуемую длину проникновения устройства в объект для более успешного проведения ВУС.

Размер: устройство имеет размер XL, достигает 21см. в длину и 4.5 см. в диаметре.

Применение: использование устройства допустимо в случае, если специалист ФИ-69 при проведении мероприятий по ВУС объекта получает статус «временно сексуально недееспособен» (к устройству прилагаются специальные крепления для имитации «естественной эрекции»). Само устройство не имеет естественной смазки, в связи с чем необходимо применять лубрикант на водной основе. Лубрикант необходимо нанести на устройство ровным слоем, а также на половые органы объекта с углублением внутрь на 3-5 см. для облегчения начала процедуры ВУС. После нанесения лубриканта требуется одной рукой придерживать объект, а второй рукой направлять устройство. Проникновение должно быть медленным во избежание нанесения повреждений объекту и продолжаться до тех пор, пока устройство не погрузится внутрь до отметки «Contained Successfully», после чего разрешается приступать к возвратно-поступательным движениям с наращиванием темпа до тех пор, пока объект не лишится сил и не будет готов к транспортировке в свою камеру содержания. В случае необходимости дополнительной стимуляции объекта, рекомендуется применять упомянутые ранее насадки. Для этого устройство необходимо извлечь из объекта, удалить лубрикант с поверхности устройства во избежание сползания насадки, после чего зафиксировать требующуюся насадку из набора и нанести поверх неё лубрикант. Далее, повторно ввести устройство в объект и продолжить мероприятия по ВУС.

Достоинства: средний размер позволяет успешно проводить мероприятия по ВУС объектов класса «Возбужденный кетер», набор специальных насадок ускоряет процесс перевода объекта в состояние «Удовлетворён», особый материал с добавлением [ДАННЫЕ УДАЛЕНЫ] позволяет устройству выдерживать сверхнормативные нагрузки, специальные крепления позволяют специалисту ФИ-69 использовать устройство по достижении состояния «временно сексуально недееспособен».

Недостатки: отсутствует вибростимуляция, отсутствует симуляция эякуляции (выброс жидкости, имитирующий мужское семя), отсутствует электростимуляция.

Рекомендации: данное устройство рекомендуется к использованию ТОЛЬКО в рамках проведения мероприятий по ВУС объектов класса «Возбужденный кетер» и категорически запрещается применять для удовлетворения сексуальных потребностей среди представителей относящихся к биологическому виду «Homo sapiens».
